## Features

- Data loading and preprocessing
- Exploratory data analysis and visualization
- Label encoding for categorical features
- Class balancing using SMOTE
- Train-test split for model evaluation
- Multiple machine learning models for comparison
- Performance evaluation using accuracy, confusion matrix, MAE, MSE, RMSE, and R2 score

## Models Used

- Logistic Regression
- Decision Tree
- Random Forest
- K-Nearest Neighbors (KNN)
- Support Vector Machine (SVM)
- Voting Classifier Ensemble
- XGBoost
- AdaBoost
- Neural Network
- Ridge Regression

- Python

- TensorFlow / Keras
- XGBoost
- Imbalanced-learn

## Workflow

1. Download and load the loan approval dataset
2. Clean and preprocess the data
3. Encode categorical variables
4. Balance the target classes using SMOTE
5. Perform visualization and correlation analysis
6. Train multiple models
7. Compare model performance using evaluation metrics
